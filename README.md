# Aplikacja do zarządzania pracownikami - zadanie rekrutacyjne Frontend Developer 03.11.2025

## Demo

[link do aplikacji](https://employee-manager-angular.netlify.app/)

## Cel projektu

Utworzenie aplikacji do zarządzania pracownikami z wykorzystaniem Angulara 20, TypeScript, RxJS.

## Technologie i narzędzia

- Angular v20.3 zoneless
- Node v22
- TypeScript v5.9
- RxJS v7.8
- HTML
- SCSS
- Jasmine, Karma
- Eslint
- Prettier

## Funkcjonalności

- Widok listy pracowników
- Dodawanie pracownika
- Edycja pracownika
- Usuwanie pracownika
- Sortowanie pracowników
- Wyszukiwanie pracowników


## Dodatkowe funkcjonalności
- Accessibility
- Motyw ciemny
